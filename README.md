# Word-Guess-Solver
AI project

## Quick Guide
Environment: Java8

Compile the code: `make` 

Run the code: `make run`

To run specific solver:
- random solver: `make random`
- dfs solver: `make dfs`
- minimax solver: `make minimax`

Clean the all the .class: `make clean`
